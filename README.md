# 🌳 TEM-4 自适应备考系统 v1.0

> 以"自适应学习路径引擎"为核心的英语专业四级（TEM-4）备考 Web 应用

## 快速开始

```bash
# 方式一：一键启动
chmod +x start.sh
./start.sh

# 方式二：手动启动
pip install flask
python app.py
```

启动后访问：**http://127.0.0.1:5000**

## 功能特性

### ⭐ 自适应规划引擎（核心）
- **知识追踪**：为每个知识点维护掌握概率，答题后实时更新
- **SM-2 间隔重复**：科学安排复习时间，对抗遗忘曲线
- **三阶段规划**：基础 → 强化 → 冲刺，根据考试倒计时自动切换
- **动态调整**：根据学习表现自动调整后续计划

### 📝 练习系统
- **词汇练习**：核心动词/名词/形容词/副词/介词/连词/搭配/近义词/学术词汇/词根词缀
- **语法练习**：时态/被动语态/虚拟语气/非谓语动词/定语从句/名词性从句/状语从句/倒装句/强调句/主谓一致/比较级/代词
- **三种模式**：综合练习 / 复习模式 / 薄弱点专项
- **即时反馈**：答题后立即显示解析

### 📊 数据仪表盘
- 五维能力雷达图（词汇/语法/听力/阅读/写作）
- 近 7 天学习趋势
- 薄弱点 TOP 5 分析
- 学习里程碑追踪
- 预估分数

## 使用流程

1. **注册账号** → 设置考试日期、每日学习时间、目标分数
2. **水平诊断** → 20 题快速评估初始能力
3. **查看仪表盘** → 了解当前水平和学习计划
4. **开始练习** → 系统智能推荐题目
5. **持续学习** → 系统自动追踪掌握度并调整计划

| 组件 | 技术 |

## 项目结构

```
tem4-prep/
├── app.py                      # Flask 主应用
├── config.py                   # 配置
├── start.sh                    # 一键启动脚本
├── engine/                     # ⭐ 自适应规划引擎
│   ├── knowledge_tracker.py    # 知识追踪
│   ├── spaced_repetition.py    # SM-2 间隔重复
│   ├── path_planner.py         # 学习路径规划
│   └── dynamic_adjuster.py     # 动态调整
├── models/
│   └── database.py             # 数据库 + 题库种子数据
├── services/                   # 业务逻辑
│   ├── auth_service.py         # 认证
│   ├── assessment_service.py   # 水平诊断
│   ├── practice_service.py     # 练习
│   └── report_service.py       # 数据报告
├── templates/                  # HTML 模板（动森风格）
├── static/css/custom.css       # 自定义样式
└── data/tem4.db                # SQLite 数据库（自动生成）
```
